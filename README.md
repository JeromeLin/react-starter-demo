# react-starter-demo

* 框架：React 0.14.0
* 工程化：webpack
* 语言：Javascript ES6(babel编译)
* 前端路由：react-router 1.0.0-rc3
* 样式表：Sass  
* 服务端：Nodejs(express)
* 前后端同构渲染页面，按需加载。


项目安装
-----------
* npm install

生产环境
-----------
* npm start
* http://localhost:3000/page1

开发环境
-----------
* npm run dev
* http://localhost:3000/page1
